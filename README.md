# VibeTunes - Mood Based Music Player

Project Overview -

This project is a web application which suggests music based on the user's mood. Instead of searching songs manually, the user can select their mood and get songs related to it.

Basic AI logic is used to map moods with suitable songs. When a user selects a song, it redirects to YouTube to play the music.

The main idea of this project is to make music selection easier and more personalized.

---

Features :-

1. User can select mood (Happy, Sad, Chill, Energetic, etc).
2. Songs are suggested based on mood selection.
3. Shows song name and artist.
4. Redirects user to YouTube for playing songs.
5. Simple and easy to use interface.
6. Basic responsive design.

---

AI / Logic Used :-

Basic AI logic is used to connect moods with songs.
The system suggests songs based on predefined mapping and simple logic.

---

Methods / Technologies used :-

1. HTML (for structure of the web page)
2. CSS (for styling and layout)
3. JavaScript (for logic and interaction)
4. React (for building UI components and managing state)

---

Project Goal

The goal of this project is to understand:

1. How to build a web application
2. How to use logic for recommendation
3. Use of JavaScript for interaction
4. Basics of React
5. Improving user experience
